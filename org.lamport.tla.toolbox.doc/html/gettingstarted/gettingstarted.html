Getting Started

The first thing you see when you start the Toolbox for the first time is the Welcome View. If you are new to the Toolbox, you should read about how to use help and how to manage the various views that the Toolbox will put on the screen.

From the welcome view, you will usually open a specification. You can also set the Toolbox preferences

For further information about TLA+ and the TLA+ tools, see the TLA+ Resources help page.
