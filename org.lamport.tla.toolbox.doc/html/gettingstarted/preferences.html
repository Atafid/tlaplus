Preferences

The Toolbox's main file menu contains a Preferences item. Clicking on it raises a dialog for setting various Toolbox preferences. The dialog presents several high-level choices. You should ignore almost all of them except TLA+ Preferences. Clicking on it provides an option that allows you to choose whether, when you close the Toolbox, it resumes from where you were (the default) or opens the Welcome View. Opening the menu item (by clicking on the + next to it reveals four choices. Click on the links below for descriptions of those choices choices.

  • TLA+ Parser Preferences
  • TLC Model Checker Preferences
  • Pretty-Printing Preferences
  • PlusCal Translator Preferences

There are two other useful items on the Preference menu that allow you to customize the Module Editor:

  • The Opening General choice and its Editors subchoice and clicking on Text Editors gives you a page with a Show line numbers option and a Colors and Fonts link at the bottom.
  • Clicking on General / Keys gives you a page with which you can change what commands are bound to what keys.
