Preferences

The Toolbox's main file menu contains a Preferences item. Clicking on it raises a dialog for setting various Toolbox preferences. The dialog presents several high-level choices. Ignore almost all of them except TLA+ Preferences. Click on the links below for help about these choices.

  • TLA+ Parser Preferences
  • Model Checking Preferences
  • Pretty-Printing Preferences
  • PlusCal Translator Preferences

The one other preference that you may find useful is the General / Keys preference, which allows you to change what commands are bound to what keys. You may find it convenient to do this for some module editor commands.
