Proofs

The current version of the TLA+ language contains several features not described in Specifying Systems.  Chief among them are constructs for writing hierarchically structured proofs.  The differences between the current and previous versions of TLA, including the proof language, are described here. 

A TLA+ proof is heirarchically structured.  The Toolbox aids reading and writing proofs by providing commands to view just the part of the structure that you want to see.  These commands are described in the subtopic Viewing Structured Proofs.

The Running TLAPS subtopic describes how to run TLAPS, the TLA+ Proof System, which is a tool for mechanically checking TLA+ proofs.  You can select the parts of a proof to be checked, and you can have the Toolbox display which steps' proofs have and have not been checked.
