Proofs

The current version of the TLA+ language, called TLA+2, contains several features not described in Specifying Systems.  Chief among them are constructs for writing hierarchically structured proofs.  TLA+2 and TLA+2 proofs are described here.  TLAPS, the TLA+ Proof System, is a tool for mechanically checking TLA+ proofs.  Soon, you will be able to run TLAPS from the Toolbox.  You will be able to select the parts of a proof to be checked, and the Toolbox will display which steps' proofs have and have not been checked.  The module editor will also provide useful commands for writing proofs.  Currently, the only sign of these features is a useless TLA Proof Manager menu.  The one feature the Toolbox now provides for readers and writers of proofs is a nice way of viewing structured TLA+ proofs.
